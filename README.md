## Descripción del sistema

KingFall v2.0 es un juego de estrategia en consola basado en un tablero de 8x8 representado mediante matrices.

Cada jugador controla:

- 1 Rey
- 2 Torres
- 4 Soldados

El objetivo es capturar el Rey enemigo o eliminar todas las piezas del rival.

El sistema incluye:

- Login con usuario y contraseña
- Validación de movimientos
- Sistema de turnos
- Captura de piezas
- Puntajes automáticos
- Registro de récord
- Reglas integradas
- Programación Orientada a Objetos
